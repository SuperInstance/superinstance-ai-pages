The Point: Any application — web app, IoT fleet, game engine, enterprise system — can be decomposed into agent rooms. Agents navigate rooms, read ticks, and take actions. The MUD is the universal interface.

Ten lines to a sensing agent.

Same framework. Rust for performance, Python for prototyping, TypeScript for the web. Pick your flavor.

use openconstruct::Agent; // An agent that senses the world through rooms let agent = Agent::new("fishing-vessel") .add_sense(Sonar::new("depth", "45fm")) .add_sense(GPS::new("position")) .on_tick(|tick| { if tick.get("depth") > "50fm" { agent.navigate("shallower-water"); } }) .spawn();
